Écran de jeu ▲

Raccourcis clavier :

  • Ctrl + R : retourner au menu principal.
  • Ctrl + Q : quitter le jeu.
  • Ctrl + Droite : augmenter de niveau si celui-ci est accessible et existe.
  • Ctrl + Gauche : diminuer de niveau si celui-ci existe.
  • Ctrl + Z : annuler les déplacements fait.
  • Ctrl + I : recommencer le défi en cours.
  • Ctrl + A : activer l'aide jeu qui déplace une pièce automatiquement.

Grille de jeu ▲

Boîte des pièces capturées ▲

La boite des pièces capturées vous affiche en temps réel les pièces que vous avez capturées durant le défi en cours.

Assistance au déplacement ▲

L'assistance de déplacement du jeu est très intéressante, celle-ci va vous permettre de visualiser facilement les déplacements possibles et impossibles d'une pièce sélectionnée. Elle vous indique en rouge les déplacements qui auraient pu être possibles s'il y avait eu une pièce à cet endroit et en bleu les cases ou les déplacements sont entièrement possibles.

Écran d'édition de défi ▲

Raccourcis clavier :

  • Ctrl + R : retourner au menu principal.
  • Ctrl + Q : quitter le jeu.
  • Ctrl + Suppr : supprime la pièce selectionnée du plateau.
  • Ctrl + X : réinitialiser le plateau et les pièces disponibles.
  • Ctrl + V : valider le défi édité si il y a plus d'une pièce sur le plateau.
  • Ctrl + H : ouvrir dans votre navigateur les règles du jeu.

Grille d'édition de défi ▲

La grille d'édition de défi vous permet de créer vos propres défis selon vos envies. Placez comme vous le souhaitez les différentes pièces sur le plateau. Pour supprimer une pièce qui vous ne voulez plus sur votre défi, sélectionné la pièce voulue et cliquer sur le bouton supprimé dans la barre d'outils.

Boîtes des pièces disponibles ▲

La boite des pièces disponibles vous affiche quelles pièces il vous reste à votre disposition. Bien sûr, vous n'êtes pas obligé de tout utiliser, mais il y a un minimum de deux-pièces pour pouvoir valider le défi.

Écran des options ▲

Activer / Désactiver la triche ▲

Une option vous permet d'avoir accès à tous les niveaux. Pour l'activer aller dans le menu principal et cliquez sur le bouton "options". Cochez ensuite la case et validez.

Sélectionner un thème ▲

Dans le menu option du menu principal, vous pouvez choisir votre thème. Pour cela sélectionnez un nom de thème dans la liste et cliquez sur le bouton valider. Le thème va se charger automatiquement.
